# Strategy Call Präsentation, Jalousien-Böttcher × Growth Unit

Präsentationsseite für den Strategy Call mit Jalousien-Böttcher.

**Live:** [boettcher.growthunit.de](https://boettcher.growthunit.de)

## Stack

Statische HTML-Seite, ohne Build-Schritt. GSAP + ScrollTrigger via CDN, Inter & JetBrains Mono via Google Fonts.

## Deployment

GitHub Pages, Custom Domain via CNAME.

DNS-Einrichtung beim Domain-Provider von growthunit.de:

```
Type:  CNAME
Name:  boettcher
Value: growthunit.github.io
TTL:   default
```
